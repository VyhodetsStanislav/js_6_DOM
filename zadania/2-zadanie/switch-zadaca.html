Задание 2

Блочная область видимости

Switch

Напеши скрипт выбора опций доставки товара

Опция хранимтся в переменной option: 1 - самовывоз, 2 - курьер, 3 - почта

Разбиваем задачу
  1. сделать переменные
  2. сделать switch 1 2 3
  3. в каждом кейсе записывать в massage строку
  4. сделать лог massage

JS код

const option = 2;
let massage = "";

switch (option) {
case 1:
massage = "самовывоз";
break;
case 2:
massage = "курьер";
break;
case 3:
massage = "почта";
break;
default:
massage = "вам перезвонит менеджер";
}

console.log(massage);
